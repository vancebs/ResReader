#ResReader  

Read the resource of Android framework and App by reflect  

Avaliable resource types are:  
- array
- bool
- color
- dimen
- drawable
- integer
- layout
- mipmap
- string
  
Maybe supported later types:  
- anim
- animator
- attr
- fraction
- id
- interpolator
- menu
- plurals
- raw
- style
- transition
- styleable
